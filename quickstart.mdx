---

## 사전 요구 사항

* Node.js 18 이상
* npm

Node.js 버전을 확인하려면:

```bash
node --version
```

Node.js를 설치하거나 업그레이드해야 한다면, [nodejs.org](https://nodejs.org)에서 다운로드하세요.

---

## Step 1: Claude Code 설치

npm으로 패키지를 전역 설치합니다:

```bash
npm install -g @anthropic-ai/claude-code
```

설치를 확인합니다:

```bash
claude --version
```

---

## Step 2: 인증

아무 디렉토리에서 `claude`를 실행하여 초기 설정을 시작합니다:

```bash
claude
```

처음 실행하면, Claude Code가 브라우저를 열고 Anthropic 계정으로 로그인하는 과정을 안내합니다. 완료되면 자격 증명이 안전하게 저장되어 이후 세션에서 재사용됩니다.

**또는**, OAuth를 사용하지 않으려면 API 키를 직접 설정할 수 있습니다:

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

<Note>
  `ANTHROPIC_API_KEY` 환경 변수가 설정되어 있으면 API 키가 우선 적용됩니다. 대화형 사용에서는 토큰 갱신을 자동으로 처리하는 OAuth를 권장합니다.
</Note>

---

## Step 3: 프로젝트로 이동

Claude Code는 현재 디렉토리 내에서 작동합니다. 작업하려는 프로젝트로 이동하세요:

```bash
cd my-project
```

---

## Step 4: 대화형 세션 시작

`claude`를 실행하여 대화형 세션을 엽니다:

```bash
claude
```

자연어로 작업을 입력할 수 있는 프롬프트가 나타납니다. Claude가 프로젝트 파일을 읽고, 명령어를 실행하고, 설명한 내용에 따라 변경 사항을 적용합니다.

**처음 시도해볼 작업 예시:**

```
> explain the structure of this codebase
> add input validation to the signup form
> write tests for the UserService class
> find all places where we catch and swallow errors
```

권한이 필요한 변경을 하기 전에, Claude가 어떤 작업을 할 계획인지 보여줍니다. 각 단계에서 승인, 거부, 또는 다른 접근 방식을 요청할 수 있습니다.

---

## Step 5: CLAUDE.md 파일 초기화

세션 내에서 `/init`을 실행하여 프로젝트용 `CLAUDE.md` 파일을 생성합니다:

```
/init
```

Claude가 저장소를 분석합니다 — 매니페스트 파일, 기존 문서, 코드 구조를 읽고 — 코드베이스에서 효과적으로 작업하는 데 필요한 명령어와 컨텍스트가 담긴 `CLAUDE.md`를 생성합니다. 생성된 파일을 커밋하기 전에 검토하고 편집해야 합니다.

<Tip>
  `CLAUDE.md`는 매 세션 시작 시 로드됩니다. 비표준 빌드 명령어, 테스팅 특이사항, 팀 고유의 코딩 컨벤션 등을 기록하기에 가장 좋은 장소입니다.
</Tip>

---

## 비대화형 명령어 실행

`-p` 플래그를 사용하면 대화형 세션에 진입하지 않고 단일 작업을 실행하고 결과를 출력합니다. 스크립팅이나 일회성 쿼리에 유용합니다:

```bash
claude -p "explain this codebase"
```

```bash
claude -p "list all TODO comments and the files they appear in"
```

```bash
claude -p "check for unused exports in src/"
```

---

## 주요 슬래시 명령어

대화형 세션 내에서 가장 유용한 슬래시 명령어는 다음과 같습니다:

| 명령어          | 설명                                                         |
| -------------- | ----------------------------------------------------------- |
| `/help`        | 사용 가능한 명령어와 키보드 단축키 표시                          |
| `/init`        | 현재 프로젝트의 `CLAUDE.md` 생성 또는 업데이트                   |
| `/memory`      | 메모리 파일 (`CLAUDE.md`, `CLAUDE.local.md`) 보기 및 편집       |
| `/permissions` | 현재 권한 모드 보기 또는 변경                                   |
| `/mcp`         | 연결된 MCP 서버 관리                                          |
| `/clear`       | 현재 대화 컨텍스트 초기화                                      |
| `/exit`        | 세션 종료                                                    |
