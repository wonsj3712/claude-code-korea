# 슬래시 명령어

슬래시 명령어는 활성 세션 중에 Claude Code 입력 프롬프트에서 `/`를 입력하여 활성화합니다. 언제든지 `/help`를 입력하면 플러그인과 스킬에 의해 추가된 명령어를 포함한 모든 사용 가능한 명령어를 확인할 수 있습니다.

## 빠른 참조

| 명령어 | 설명 |
|--------|------|
| `/init` | 프로젝트의 `CLAUDE.md` 파일과 선택적 스킬/훅을 생성합니다 |
| `/memory` | Claude 메모리 파일을 편집합니다 (전역, 프로젝트, 로컬) |
| `/config` | 설정 패널을 엽니다 |
| `/hooks` | 도구 이벤트에 대한 훅 설정을 확인합니다 |
| `/mcp` | MCP 서버를 관리합니다 — 활성화, 비활성화, 재연결 |
| `/permissions` | 도구에 대한 허용 및 거부 규칙을 관리합니다 |
| `/plan` | 계획 모드를 활성화하거나 세션 계획을 관리합니다 |
| `/model` | 현재 세션의 AI 모델을 설정합니다 |
| `/commit` | AI가 생성한 메시지로 git 커밋을 생성합니다 |
| `/review` | 풀 리퀘스트를 검토합니다 |
| `/skills` | 사용 가능한 스킬을 나열합니다 |
| `/compact` | 대화 기록을 요약하여 컨텍스트 사용량을 줄입니다 |
| `/clear` | 대화 기록을 지우고 컨텍스트를 해제합니다 |
| `/help` | 도움말과 사용 가능한 명령어를 표시합니다 |
| `/login` | Anthropic 계정에 로그인하거나 계정을 전환합니다 |
| `/logout` | Anthropic 계정에서 로그아웃합니다 |

## 상세 명령어 설명

### 프로젝트 및 메모리 명령어

**`/init`**

코드베이스를 분석하고 인터뷰를 진행한 후 프로젝트 구성 파일과 선택적 스킬/훅을 생성합니다.

**`/memory`**

세 가지 메모리 범위의 편집기를 엽니다:
- 전역 (`~/.claude/CLAUDE.md`)
- 프로젝트 (루트의 `CLAUDE.md`)
- 로컬 (루트의 `CLAUDE.local.md`, gitignore 처리됨)

### 설정 명령어

**`/config`** (별칭: `/settings`)

모델 환경설정, 테마, 상세 출력 모드 설정을 위한 구성 패널을 엽니다.

**`/hooks`**

활성 훅 설정을 표시합니다. 훅을 생성/수정하려면 `/init`을 사용하거나 `.claude/settings.json`을 직접 편집해야 합니다.

**`/mcp` [enable|disable [서버명]]**

MCP 서버를 관리합니다. 인수:
- `enable` — 모든 MCP 서버 활성화
- `enable <서버명>` — 특정 서버 활성화
- `disable` — 모든 MCP 서버 비활성화
- `disable <서버명>` — 특정 서버 비활성화
- `reconnect <서버명>` — 특정 서버 재연결

**`/permissions`** (별칭: `/allowed-tools`)

`Bash(git:*)`와 같은 glob 스타일 패턴을 사용하여 허용/거부 규칙을 설정하는 권한 패널을 엽니다.

### 세션 관리 명령어

**`/plan` [open|<설명>]**

계획 모드를 관리합니다. 인수:
- (없음) — 계획 모드 토글
- `open` — 현재 계획 열기
- `<설명>` — 설명으로 새 계획 생성

**`/model` [모델]**

세션의 AI 모델을 설정합니다. 사용 가능한 값: `sonnet`, `opus`, `haiku`, 또는 `claude-sonnet-4-6`과 같은 특정 모델 ID

**`/compact` [지시사항]**

대화 기록을 요약합니다. 선택적으로 특정 요약 지시사항을 전달할 수 있습니다.

**`/clear`** (별칭: `/reset`, `/new`)

전체 대화 기록을 지우고 같은 디렉토리에서 새로 시작합니다.

**`/skills`**

현재 세션에서 사용 가능한 모든 스킬을 나열합니다.

### Git 명령어

**`/commit`**

기존 저장소 스타일과 안전 규칙을 따르는 AI 생성 커밋 메시지로 git 커밋을 생성합니다.

**`/review` [PR 번호]**

GitHub CLI를 통해 구조화된 코드 리뷰를 제공합니다. `gh`가 설치되어 있고 인증되어 있어야 합니다.

### 계정 및 도움말

**`/help`**

내장, 스킬, 플러그인 명령어를 포함한 모든 사용 가능한 슬래시 명령어를 나열합니다.

**`/login`**

Anthropic 계정에 로그인하거나 계정 간 전환을 합니다.

**`/logout`**

Anthropic 계정에서 로그아웃합니다.

## 사용자 정의 스킬

`.claude/skills/<스킬명>/SKILL.md`에 사용자 정의 스킬을 생성하면 `/<스킬명>` 명령어로 호출할 수 있습니다.

예시:
- `/verify`
- `/deploy staging`
- `/fix-issue 123`
