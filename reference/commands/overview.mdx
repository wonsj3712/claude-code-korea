# 명령어 개요

Claude Code는 두 가지 범주의 명령어를 제공합니다: 실행 시 전달하는 CLI 플래그와 세션 내에서 입력하는 슬래시 명령어입니다.

Claude Code에는 두 가지 범주의 명령어가 있습니다:

* **CLI 플래그** — 터미널에서 Claude를 시작할 때 전달하는 옵션입니다(예: `claude --model sonnet`). 세션이 시작되기 전에 설정을 구성합니다.
* **슬래시 명령어** — 활성 세션 내에서 입력하는 텍스트 명령어입니다(예: `/help`). 세션이 실행 중인 동안 Claude의 동작을 제어합니다.

## 도움말 보기

```bash
# 모든 CLI 플래그 표시
claude --help

# 세션 내에서 슬래시 명령어 목록 확인
/help
```

`/help`는 플러그인과 스킬에 의해 추가된 명령어를 포함하여 현재 세션에서 사용 가능한 모든 슬래시 명령어를 나열합니다.

## 두 가지 유형의 명령어

| 유형           | 사용 시점                                                                     | 예시                                                   |
| -------------- | ----------------------------------------------------------------------------- | ------------------------------------------------------ |
| CLI 플래그     | 세션 시작 시 구성 — 모델, 출력 형식, 권한 모드 설정                           | `claude --permission-mode acceptEdits "fix the tests"` |
| 슬래시 명령어  | 실행 중인 세션과 상호작용 — 메모리 관리, 모델 전환, 코드 커밋                 | `/commit`                                              |

CLI 플래그는 한 번 사용되며 세션 중간에 변경할 수 없습니다(`/model`과 `/permissions`는 예외로, 실행 중인 구성을 직접 변경할 수 있습니다).

## CLI 플래그

`claude` 뒤에 직접 플래그를 전달합니다:

```bash
claude [플래그] [프롬프트]
```

```bash
# 비대화형: 응답을 출력하고 종료
claude -p "summarize this file" < README.md

# 세션의 모델 설정
claude --model opus

# 모든 파일 편집을 자동으로 승인
claude --permission-mode acceptEdits
```

## 슬래시 명령어

세션 내의 입력 프롬프트에서 슬래시 명령어를 입력합니다:

```
/명령어 [인수]
```

```
/init
/compact summarize only the last three tasks
/model claude-opus-4-5
```

## 키보드 단축키

| 키            | 동작                                                              |
| ------------- | ----------------------------------------------------------------- |
| `Ctrl+C`      | 현재 응답 중단 (Claude가 응답 중간에 멈춤)                        |
| `Ctrl+D`      | Claude Code 종료                                                  |
| `Ctrl+L`      | 터미널 화면 지우기 (대화 기록은 삭제되지 않음)                    |
| `Up` / `Down` | 입력 기록 탐색                                                    |
| `Tab`         | 슬래시 명령어 이름 자동완성                                       |
| `Escape`      | 진행 중인 권한 프롬프트 취소                                      |

## 하위 명령어

| 하위 명령어         | 설명                                          |
| ------------------- | --------------------------------------------- |
| `claude mcp`        | MCP 서버 구성 및 관리                          |
| `claude mcp serve`  | Claude Code를 MCP 서버로 시작                  |
| `claude doctor`     | 설치 및 구성 문제 진단                         |
| `claude update`     | Claude Code를 최신 버전으로 업데이트           |

```bash
claude mcp --help
claude doctor
```
