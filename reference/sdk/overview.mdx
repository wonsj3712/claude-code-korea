# SDK 개요

Claude Code SDK는 외부 애플리케이션에 Claude를 임베딩하기 위한 제어 프로토콜입니다 — IDE, 자동화 스크립트, CI/CD 파이프라인, 또는 서브프로세스를 생성하고 stdin/stdout으로 통신할 수 있는 모든 호스트에서 사용할 수 있습니다.

직접적인 라이브러리 API를 사용하는 대신, 실행 중인 `claude` 프로세스와 stdin/stdout 스트림을 통해 구조화된 JSON 메시지로 통신합니다.

## 운영 흐름

SDK는 네 가지 순차적 단계로 작동합니다:

1. **프로세스 실행**: `--output-format stream-json`과 `--print` 플래그로 `claude`를 시작하여 비대화형 모드로 실행합니다
2. **초기화**: `subtype: "initialize"`가 포함된 제어 요청을 전송하여 세션을 구성합니다
3. **스트림 처리**: stdout에서 줄바꿈으로 구분된 JSON을 읽어 다양한 메시지 유형을 처리합니다
4. **대화**: stdin에 `SDKUserMessage` 객체를 작성하여 대화를 이어갑니다

## 출력 형식 옵션

`--output-format` 플래그를 통해 세 가지 형식을 사용할 수 있습니다:

| 형식 | 설명 |
|------|------|
| `text` | 일반 텍스트 응답만 출력 (대화형 모드 기본값) |
| `json` | 완료 시 단일 JSON 객체 출력 |
| `stream-json` | 줄바꿈으로 구분된 JSON 스트림 (SDK 사용 시 필수) |

## 메시지 프로토콜 아키텍처

양방향으로 두 가지 엔벨로프 유형이 전달됩니다:

### SDKControlRequest (호스트 → CLI)

`type`, `request_id`, 그리고 작업 하위 유형이 포함된 `request` 페이로드를 포함합니다.

### SDKControlResponse (CLI → 호스트)

`type`, 성공/오류 상태가 포함된 `response` 객체와 에코된 `request_id`를 반환합니다.

## 초기화 설정

초기화 요청은 다음을 지원합니다:

- **`systemPrompt`**: 기본 지침을 대체합니다
- **`appendSystemPrompt`**: 기존 프롬프트에 추가합니다
- **`hooks`**: 특정 이벤트에 대한 콜백 핸들러를 등록합니다
- **`agents`**: 설명, 프롬프트, 모델 할당이 포함된 사용자 정의 서브에이전트를 정의합니다
- **`sdkMcpServers`**: 프로세스 내 MCP 서버를 연결합니다

응답에는 사용 가능한 명령어, 에이전트, 모델, 출력 스타일, 계정 정보가 포함됩니다.

## 사용자 메시지

Anthropic API 호환 메시지 페이로드가 포함된 `SDKUserMessage` 객체를 전송합니다. 메시지는 도구 대화를 이어가기 위해 부모 도구 사용 ID를 참조할 수 있고, 추적을 위한 선택적 UUID를 포함할 수 있으며, 스케줄링 우선순위(`now`, `next`, `later`)를 지정할 수 있습니다.

## 스트림 메시지 유형

CLI는 다양한 메시지 유형을 출력합니다:

| 메시지 유형 | 설명 |
|-------------|------|
| `system` | 세션 초기화 및 상태 업데이트 |
| `assistant` | 도구 사용 블록이 포함된 모델 응답 |
| `stream_event` | 점진적 렌더링을 위한 부분 스트리밍 토큰 |
| `tool_progress` | 장시간 실행되는 도구의 상태 |
| `result` | 비용과 소요 시간 지표가 포함된 최종 턴 요약 |

## 고급 제어 작업

초기화 외에 사용 가능한 작업들:

| 작업 | 설명 |
|------|------|
| `interrupt` | 현재 실행 중단 |
| `set_permission_mode` | 권한 수준 조정 |
| `set_model` | 세션 중간에 모델 전환 |
| `mcp_status` | MCP 서버 연결 상태 확인 |
| `get_context_usage` | 토큰 사용량 분석 조회 |
| `rewind_files` | 특정 메시지의 변경 사항 되돌리기 |

## TypeScript SDK 함수

`@anthropic-ai/claude-code` 패키지에서 내보내는 주요 함수:

| 함수 | 설명 |
|------|------|
| `query()` | 프롬프트 또는 비동기 이터러블을 받는 기본 진입점 |
| `listSessions()` | 저장된 세션 메타데이터 조회 |
| `getSessionMessages()` | 트랜스크립트 파일 파싱 |
| `forkSession()` | UUID 재매핑과 함께 대화 분기 |
| `renameSession()` | 세션 메타데이터 업데이트 |
| `tagSession()` | 세션에 태그 부여 |

## 실용적 활용 사례

### IDE 통합

사용자 정의 시스템 프롬프트로 컨텍스트를 설명하고, 훅 콜백으로 파일 편집을 가로채어 네이티브 UI에 표시합니다.

### CI/CD 자동화

일회성 파이프라인 작업에 `--output-format json --print`를 사용하여 결과를 프로그래밍 방식으로 파싱합니다.

### 헤드리스 에이전트

지속적인 백그라운드 데몬을 위해 TypeScript `query()`를 스트리밍과 함께 활용합니다.
