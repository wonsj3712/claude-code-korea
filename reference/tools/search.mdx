## Glob 도구

Glob 도구는 경로 패턴을 매칭하여 파일을 찾습니다. 표준 glob 구문을 지원하며, `*`(세그먼트 내 문자), `**`(여러 경로 세그먼트), `?`(단일 문자)를 사용할 수 있습니다. 결과는 수정 시간순으로 정렬되며 최대 100개 파일로 제한되고, 작업 디렉토리 기준의 상대 경로로 표시됩니다.

## Grep 도구

Grep 도구는 ripgrep을 기반으로 정규식을 사용하여 파일 내용을 검색합니다.

### 핵심 파라미터

- 전체 정규식 구문을 지원하는 패턴 매칭
- 선택적 경로 또는 glob 필터링
- `type` 파라미터를 통한 파일 유형 제한

### 출력 모드

- `"files_with_matches"`: 파일 경로만 반환 (기본값)
- `"content"`: 선택적 컨텍스트와 함께 일치하는 줄을 반환
- `"count"`: 파일별 일치 횟수를 반환

### 컨텍스트 옵션

- `-A`: 일치 항목 뒤에 표시할 줄 수
- `-B`: 일치 항목 앞에 표시할 줄 수
- `context`: 일치 항목 앞뒤 대칭 줄 수
- `-i`: 대소문자 구분 없는 검색

### 고급 기능

- `multiline` 모드: 여러 줄에 걸친 패턴 매칭
- `head_limit`와 `offset` 파라미터: 페이지네이션 지원
- 줄 번호 표시: content 모드에서 기본적으로 활성화

<Note>
ripgrep은 기본적으로 `{`와 `}`를 리터럴 문자로 취급하므로, 패턴 매칭 시 이스케이프가 필요합니다.
</Note>

버전 관리 디렉토리(`.git`, `.svn` 등)는 자동으로 검색에서 제외됩니다.
