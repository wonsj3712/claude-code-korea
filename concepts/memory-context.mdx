Claude Code는 마크다운 파일을 사용한 계층적 메모리 시스템을 구현하여, 여러 조직 수준에서 영속적이고 프로젝트 인식이 가능한 지침을 제공합니다.

## 4단계 메모리 계층

### 관리형 메모리 (최저 우선순위)

- **위치:** `/etc/claude-code/CLAUDE.md`
- **범위:** 시스템 전체, 관리자가 설정
- **특성:** 머신의 모든 사용자에게 적용됩니다. 정책을 통해 강제 적용할 경우 재정의할 수 없습니다.

### 사용자 메모리

- **위치:** `~/.claude/CLAUDE.md` 및 `~/.claude/rules/*.md`
- **범위:** 모든 프로젝트에 적용되는 개인 전역 지침
- **사용 사례:** 개인 선호 설정, 코드 스타일, 언어 기본값, git 설정

### 프로젝트 메모리

- **위치:** `CLAUDE.md`, `.claude/CLAUDE.md`, `.claude/rules/*.md` (상위 디렉토리에서도 확인)
- **범위:** 소스 컨트롤에 포함되는 팀 공유 지침
- **사용 사례:** 프로젝트 규칙, 아키텍처 노트, 테스트 명령

### 로컬 메모리 (최고 우선순위)

- **위치:** `CLAUDE.local.md` (상위 디렉토리에서도 확인)
- **범위:** 비공개 프로젝트별 재정의
- **참고:** `.gitignore`에 추가해야 합니다

## 파일 검색 알고리즘

시스템은 현재 작업 디렉토리에서 루트까지 상위로 이동하며, 각 수준에서 파일을 수집합니다. 나중에 로드된 파일이 컨텍스트 윈도우에서 더 높은 우선순위를 갖습니다. 이 검색 순서는 낮은 우선순위 파일이 조립된 컨텍스트에서 먼저 나타나도록 보장합니다.

검색은 `utils/claudemd.ts`의 `getMemoryFiles()`를 사용하며, 대화 기간 동안 메모이제이션됩니다.

## @include 지시어

메모리 파일은 `@` 표기법을 사용하여 다른 파일을 참조할 수 있습니다:

```markdown
@filename
@./relative/path
@~/home/path
@/absolute/path
```

**주요 규칙:**
- 코드 블록 내의 경로는 무시됩니다
- 순환 참조는 감지되어 건너뜁니다
- 존재하지 않는 파일은 조용히 무시됩니다
- 최대 포함 깊이: 5단계
- 텍스트 기반 파일만 포함됩니다 (`.md`, `.ts`, `.py`, `.json` 등)
- 외부 include는 기본적으로 명시적 승인이 필요합니다

## .claude/rules/*.md — 세분화된 규칙

파일은 `.claude/rules/` 디렉토리에 분리할 수 있으며, 선택적으로 경로 범위 지정 프론트매터를 사용합니다:

```markdown
---
paths:
  - "src/api/**"
  - "src/services/**"
---
```

이렇게 하면 일치하는 파일 패턴에만 규칙을 제한하여 컨텍스트를 가볍게 유지할 수 있습니다.

## 파일 크기 제한

최대 권장 크기: **40,000자** (`MAX_MEMORY_CHARACTER_COUNT`). 이 제한을 초과하면 경고가 발생하고 잘릴 수 있습니다.

## 컨텍스트 우선순위

메모리 파일은 다음과 같은 명시적 지침과 함께 조립됩니다: *"코드베이스 및 사용자 지침이 아래에 표시됩니다. 반드시 이 지침을 준수하세요. 중요: 이 지침은 모든 기본 동작을 재정의하며, 작성된 그대로 정확히 따라야 합니다."*

## 사용 가이드라인

- **관리형 메모리:** 조직 정책, 보안 가드레일
- **사용자 메모리:** 프로젝트 전반에 걸친 개인 선호 설정
- **프로젝트 메모리:** 팀 규칙, 빌드 명령, 아키텍처 결정
- **로컬 메모리:** 개인 프로젝트별 재정의

## /memory 명령

메모리 파일 편집기를 열어, 로드된 파일을 표시하고, 직접 편집을 가능하게 하며, 저장 시 컨텍스트를 다시 로드합니다.

## 메모리 로딩 비활성화

| 방법 | 효과 |
|------|------|
| `CLAUDE_CODE_DISABLE_CLAUDE_MDS=1` | 모든 메모리 파일 로딩을 비활성화합니다 |
| `--bare` 플래그 | CWD 탐색을 통한 자동 검색을 건너뜁니다 |
| `claudeMdExcludes` 설정 | 일치하는 glob 패턴을 제외합니다 |
